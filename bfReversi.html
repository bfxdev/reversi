Game rules

The game of Reversi, also known as Othello, is played on a board of 8x8 cells (or squares). Players use discs thare are black on one side, and white on the other. The game evolves by reversing the discs. TBD

About this game

This game is a Web Widget. I wrote it to learn basics of CSS and Javascript. I created the program form scratch using Aptana Studio with the Nokia WRT plugin. The layout has been designed primarily for the 5800, but should adapt to some other screen resolutions.

The design and parts of the code are derived from the Checkers game by moubail.com.
